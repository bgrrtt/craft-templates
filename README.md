# Craft Templates

## Install
#### 1. Create directory and initialize git. (or cd into your craft/ folder)
```
mkdir TMP_HOLD && cd $_ && git init
```

#### 2. Pull repository.
```
git pull https://github.com/bgrrtt/craft-templates.git
```

#### 3. Pull plugins.
```
git submodule update --init --recursive
```
